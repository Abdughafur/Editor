# 🖼️ Simple Image Editor

A lightweight and modern browser-based image editor built with **HTML, CSS, and JavaScript**.  
It allows users to upload, edit, and download images directly in the browser — no backend required.

---

## ✨ Features

- 📤 Upload image (file picker + drag & drop)
- 🔄 Rotate image (0° - 360°)
- 🌗 Adjust brightness
- ⚖️ Adjust contrast
- 🎨 Adjust saturation
- ↩️ Undo / Redo system
- 🔁 Reset edits
- 💾 Download edited image as PNG
- 🖱️ Drag to move image on canvas
- 📱 Responsive design (works on mobile & desktop)

---

## 🧠 Tech Stack

- HTML5
- CSS3 (modern UI + responsive layout)
- Vanilla JavaScript
- Canvas API

---

## 🚀 How to Use

1. Open https://abdughafur.github.io/Editor/ link in your browser
2. Click **Upload Image** or drag & drop an image
3. Use sliders to edit:
   - Rotate
   - Brightness
   - Contrast
   - Saturation
4. Drag image to reposition it
5. Click **Download** to save your result
